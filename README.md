# skill-demand-salary-intelligence

Project Overview:

This project analyzes LinkedIn job postings to provide insights on in-demand skills, salary trends, and job demand by location. It answers the critical question:

  > Which skills are in demand right now, and how much salary do they offer?

By leveraging real-world job data, this project showcases a full data analyst workflow from data cleaning and transformation to visualization and insight generation.

---

Business Problem:

Recruiters and job seekers often lack clear, up-to-date information on:

The most sought-after technical skills

Salary expectations associated with these skills

Geographic distribution of job opportunities

Salary variations across job roles

This platform fills that gap by providing actionable insights for workforce planning and career decisions.

---

Tools & Technologies:

Python — Data cleaning and transformation

Pandas — Data manipulation

Power BI — Dashboard and visualization

Excel — Data validation and exploration

GitHub — Version control and project hosting

---

Dataset:

Source: LinkedIn job postings (USA, Canada, Africa)

Key fields: Job Title, Job Description, Salary, Location, Skills


The dataset already contains an extracted skills column listing relevant skills for each job posting.


---

Project Workflow:

1. Combined multiple LinkedIn CSV job datasets into a single dataset

2. Cleaned and standardized job and salary data with Python

3. Used the existing skills column for skill analysis

4. Transformed skills data for analysis in Power BI

5. Created interactive dashboards to visualize skill demand, salary trends, and job locations

---

Key Insights:

Top Skills: Python and SQL dominate job postings, highlighting their demand

Salary Analysis: Roles requiring SQL and Python tend to offer higher salaries

Geographic Trends: Major cities show a concentration of data analyst jobs

Role vs Salary: Salary varies significantly by job title
